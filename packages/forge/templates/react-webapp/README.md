# React Web App

A React 18 web application with Vite and TypeScript.

## Build

```bash
npm install
npm run dev
```

## Structure

- `src/App.tsx` — Main application component
- `src/main.tsx` — Entry point
- `index.html` — HTML shell
- `vite.config.ts` — Vite configuration

## Development

All source files should use the `.beta` suffix during development.
