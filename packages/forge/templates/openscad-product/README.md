# OpenSCAD Parametric Product

A parametric OpenSCAD product template with configurable dimensions.

## Usage

1. Open `main.scad` in OpenSCAD
2. Adjust parameters at the top of the file
3. Preview (F5) and render (F6)
4. Export to STL for 3D printing (File → Export → STL)

## Structure

- `main.scad` — Parametric module with example configuration

## Parameters

- `width` — X dimension in mm
- `depth` — Y dimension in mm
- `height` — Z dimension in mm
- `wall` — Wall thickness in mm
- `corner_radius` — Rounded corners radius
