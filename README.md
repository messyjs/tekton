## Features

| Feature | Source | Description |
|---------|--------|-------------|
| SCP Protocol | OpenMythos | Structured Caveman Protocol for inter-agent communication |
| 3-Tier Compression | Caveman | Lite/Compact/Full compression to stay in context |
| Adaptive Routing | OpenMythos | Complexity-based model selection (fast вҶ” deep) |
| Learning Loop | Hermes | Evaluation вҶ’ skill extraction вҶ’ context hygiene |
| Multi-Platform | Gate | Telegram, Discord, Slack, WhatsApp, +6 more adapters |
| Voice I/O | Voice | STT localвҶ’GroqвҶ’OpenAI, TTS edgeвҶ’ElevenLabsвҶ’OpenAI |
| Document Intelligence | Docling | PDF, DOCX, PPTX, XLSX, images вҶ’ Markdown with OCR and tables |
| Web Dashboard | Dashboard | 11-page React SPA with dark theme |
| Training Orchestration | ML-Ops | QLoRA, ternary BitNet, GRPO reasoning |
| Fallback Chains | OpenMythos | Automatic model failover on errors |

## Mobile App

Tekton also includes a Flutter-based mobile app (`mobile/`) providing:

- **Multi-backend chat** вҖ” OpenAI, Anthropic, Ollama, custom endpoints
- **On-device inference** вҖ” llama.cpp FFI for Gemma models
- **Progressive install** вҖ” Chat-only вҶ’ Engine вҶ’ Models вҶ’ Agents
- **Multi-agent orchestration** вҖ” Director (auto-route) + User (manual) modes
- **Tool system** вҖ” Filesystem, web search, calculator, doc analysis, code exec
- **Memory** вҖ” Persistent cross-conversation memory with forgetting curve
- **Server mode** вҖ” Act as an OpenAI-compatible API for other devices

See [mobile/README.md](mobile/README.md) for details.

## Architecture

8 packages in a monorepo + Flutter mobile app:

```
tekton/
в”ңв”Җв”Җ packages/core/          # Foundation: SCP, routing, compression, memory
в”ңв”Җв”Җ packages/hermes-bridge/ # Learning loop: evaluation, skill extraction
в”ңв”Җв”Җ packages/tools/          # Tool execution: 10 toolsets (incl. Docling)
в”ңв”Җв”Җ packages/cli/            # Terminal interface: 23+ commands
в”ңв”Җв”Җ packages/gateway/        # Messaging: 10 platform adapters
в”ңв”Җв”Җ packages/voice/          # Voice I/O: STT/TTS/recording
в”ңв”Җв”Җ packages/dashboard/       # Web UI: 11 pages (incl. Documents), REST API
в”ңв”Җв”Җ packages/ml-ops/          # Training: QLoRA, ternary, GRPO
в”ңв”Җв”Җ packages/docling-service/ # Document parsing sidecar (Python)
в””в”Җв”Җ mobile/                   # Flutter app: Android + Desktop AI platform
```

See [docs/ARCHITECTURE.md](docs/ARCHITECTURE.md) for full architecture details.
